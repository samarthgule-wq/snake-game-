# snake-game-
Snake Game – Modular JavaScript Implementation

A classic Snake Game built using modern JavaScript (ES Modules) with a clean and modular architecture. The project separates game logic, rendering, and input handling to maintain scalability and code readability.

The application uses HTML5 Canvas for rendering the game, lightweight CSS for styling, and a minimal Node.js server for running and testing the project. The codebase follows a structured folder organization, making it easy to understand, extend, and maintain.

 **Key Features**

Modular JavaScript ES Modules architecture
Canvas-based rendering for smooth gameplay
Separate modules for game logic, rendering, and input handling
Lightweight Node.js server setup
Organized project structure for maintainability
Simple test runner for validating functionality

**Tech Stack**

*_JavaScript (ES Modules)
*_HTML5 Canvas
*_CSS
*_JSON (Project configuration)
*_Node.js

📂 Project Structure

public/
 ├── snake/        # Game modules (logic, rendering, input)
 ├── index.html    # Main game page
 └── styles.css    # Game styling

server.mjs         # Local development server
test/run-tests.js  # Test runner
package.json       # Project configuration

The purpose of this project is to demonstrate modular JavaScript architecture, game development fundamentals, and clean project structuring using modern web technologies.
